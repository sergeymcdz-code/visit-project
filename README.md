VisCards — API Backend

Backend API для мобильного приложения хранения визиток
(Android & iOS)

Проект реализует:
 • регистрацию по номеру телефона
 • хранение личных и рабочих визиток
 • импорт визиток по QR и фото (OCR)
 • управление профилем
 • обратную связь
 • защиту приложения PIN-кодом


Online API documentation (Swagger)

Документация API доступна по ссылке:

https://sergeymcdz-code.github.io/visit-project/swagger.html

Swagger сгенерирован из официального OpenAPI-контракта и полностью описывает backend.

OpenAPI спецификация

Официальная спецификация API:
 • JSON
openapi_viscards_v1_2.json
 • YAML
openapi_viscards_v1_2.yaml

Используется для:
 • генерации Swagger UI
 • интеграции frontend и mobile-клиентов
 • контрактного тестирования

Postman

Для демонстрации и ручного тестирования API используется коллекция:

viscards_api_v1_2.postman_collection.json

Как использовать:
 1. Импортировать файл в Postman
 2. Указать переменную baseUrl
 3. Получить токен через Auth → Verify SMS
 4. Вызывать любые методы API
